 Quiz Web App 🎯

A simple quiz web application built using HTML, CSS, and JavaScript.

 🚀 Features

* Multiple choice questions
* Instant answer checking
* Score tracking
* Simple and clean UI

 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

📂 Project Structure

* index.html → Main page
* style.css → Styling
* script.js → Logic

 ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
